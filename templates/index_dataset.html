1단계 동영상으로 학습 데이터 만들기

동영상을 올리면 AI가 자동으로 장면을 잘라내고, 물체를 찾아 파란 박스(인식 영역)로 표시합니다.
표시된 박스가 정확한지 확인하고, 잘못된 부분은 직접 수정해 주세요. 정확할수록 학습도 잘 됩니다.

① 동영상 업로드
mp4, avi, mov 파일 선택 후 "업로드 & 처리" 클릭
② 동영상 선택
"작업 목록"에서 업로드한 동영상 클릭
③ 장면(프레임) 선택
"프레임 목록"에서 확인할 장면 클릭 → 오른쪽에 이미지 표시
④ 영역(박스) 수정
"박스 편집 모드" ON → 박스 드래그/크기조정 → 저장
